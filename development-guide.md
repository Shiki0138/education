## 🐛 トラブルシューティング

### よくある問題

**1. サーバーが起動しない**
```bash
# ポート確認
lsof -i :8081

# 別ポートで起動
python3 -m http.server 8082
```

**2. JavaScriptエラー**
- ブラウザのConsoleでエラー確認
- ファイルの読み込み順序をチェック
- キャッシュクリア（Cmd+Shift+R）

**3. データが保存されない**
- LocalStorageの容量確認
- ブラウザのプライベートモード確認
- セキュリティ設定確認
